一、 数据类型

基本数据类型

Java的基本数据类型可以分为两大类：boolean类型和数值类型。数值类型分为整型和浮点型，整型中的字符型往往被单独对待。（string类型不是基本类型，是一个引用类型。）

1. 整型

byte：一个byte型整数在内存里占8位，-128-127
short：一个short类型占16位，-32768-32767
int：一个int类型占32位，-2147483648-2147483647
long：占64位，负的2的63次幂到正的2的63次幂-1；

使用一个巨大的整数常量（超出了int的范围），Java不会自动把这个整数常量当成long类型来处理，应该在整数常量后加l或L作为后缀。（推荐使用L，因为l很容易与1搞混。）

Java中整数常量有三种表达方式：10进制、8进制、16进制。8进制数以O开头，6进制数以OX或Ox开头，如下：

int var1=O13;

int var2=Ox13;

int var3=OXaF;

2. 字符型

字符型表示单个字符，字符常量必须用单引号（’’）括起来。Java语言使用16位的Unicode编码集作为编码方式，所以Java程序支持各种语言的字符。

字符常量有三种表示方式：

  • 直接通过单个字符来指定，如’A’
  • 通过转义字符来表示特殊字符常量，如’\n’
  • 直接使用Unicode值来表示字符常量，格式是：’\uXXXX’，其中XXXX表示一个16进制的整数

    Java常用的转义字符

    \b 退格符

    \n 换行符

    \r 回车符

    \t 制表符

    \” 双引号

    \’ 单引号

    \ 反斜线

    字符常量也可以用16进制编码方式来表示，范围是0000-FFFF，共65535个字符，其中前256个字符和ASCII码中的字符完全重合。

    关于char变量的赋值方式：

    char zhong;

    zhong=’a’;

    zhong=20014;范围：0-65535

    zhong=’\u9999’;

    zhong=20012;

    int var=zhong;这种用法是可以的；

    String是一个类，可以通过查阅API文档来使用。在Java语言中单引号、双引号和反斜线都有特殊的用途，如果一个字符串中包含这些特殊字符，应该使用转义字符的表示形式。如要写：”c:\codes”，在代码中应呈现为：”c:\codes”。

    3.浮点型

    Java的浮点类型有两种：float和double，一个float类型占4个字节，32位，一个double占8个字节，64位。浮点类型可以用科学计数形式表示，但只有浮点类型可以用科学计数法表示。Java默认浮点类型是double，如果希望Java把一个数用float来处理，在数后面加F或f。

    Java提供了三个特殊的浮点值：正负无穷大、非数。正无穷大用POSITIVE-INFINITY表示；负无穷大用NEGATIVE-INFINITY表示；非数用NaN表示。所有正无穷大都相等，所有负无穷大都相等，但是所有NaN都不相等。

    4.布尔型

    boolean用来表示逻辑上的真假。在Java中boolean类型的值只能用true和false来表示，不能有0或非0来代表。其他基本数据类型的值也不能转换成boolean类型。boolean类型的值或变量主要用来做旗标进行流程控制。

    基本类型的类型转换

    Java里支持把一个表数范围小的类型的数值直接赋给一个表数大的类型的变量，系统自动完成类型转换。

    强制类型转换：使用格式为（targetType）var；类似于把一个大瓶子里的水倒入一个小瓶子，大瓶子里的水多就会发生溢出，造成数据丢失，成为缩小转换。

    表达式类型的自动提升：当一个算术表达式中包含多个基本类型时，整个算术表达式的数据类型将发生自动提升。Java定义如下的自动提升规则：

  • 所有byte、short、char型都将被提升为int型；
  • 整个算术表达式的数据类型自动提升到与表达式中最高等级操作数同样的类型。

    直接量

    直接量指在程序中通过源代码直接指定的值，例如int a=5。

    1. 直接量的类型

    并不是所有数据类型都可以指定直接量，只有三种类型可以：基本类型、字符串类型、null类型。

    运算符

    运算符是一种特殊的符号，用以表示数据的运算、赋值和比较等。Java使用运算符将一个或多个操作数连缀成执行语句，用以实现特定功能。Java语言中运算符可以分为以下几种：

  • 算术运算符
  • 比较运算符
  • 赋值运算符
  • 逻辑运算符
  • 位运算符
  • 类型相关运算符

    1. 算术运算符

    Java支持所有的基本算术运算符：加、减、乘、除和求余等。Java没有提供复杂的运算，如需完成开方、乘方等运算，可借助于java.lang.Math类的工具完成复杂的数学运算。Math类下包含了丰富的静态方法，用于完成各种复杂的数学运算。

    2. 位运算符

    Java支持的位运算符有七个：

  • &：按位与
  • |：按位或
  • ~：按位非
  • ^：按位异或
  • <<：左位移运算符

  • ：右位移运算符

  • ：无符号右移运算符

    位移运算中，左位移后面多出来的空位补0，前面位移出去的位数截掉。右位移前面多出来的空位补符号，后面多出去的截掉。

    3. 三目运算符

    三目运算符只有一个？：。使用方法如下：

    string strone=5>3?”5大于3”:”5不大于3”;

    当条件满足时，返回第二个操作数的值，不满足则返回第三个操作数的值。
