Java笔记（二）流程控制和数组

April 11 2019

分支结构

Java提供两种常见的分支结构：if语句和switch语句。其中if语句使用布尔表达式或布尔值作为分支条件来进行分支控制，switch语句则用于对多个整型值进行匹配，从而实现分支。

1. switch分支语句

switch语句中控制表达式的类型只能是byte、short、char和int。从满足条件的case开始执行，除非遇到break，否则一直向后执行完所有后面的case以及default。

循环结构

一般的循环语句可能包含四部分：

  • 初始化语句：完成初始化工作，在循环开始之前执行
  • 循环条件：一个boolean表达式，决定是否执行循环体
  • 循环体：循环的主体，将被重复执行
  • 迭代语句：这个部分在每次循环结束后对循环条件中的变量进行控制，使循环在合适的时候结束。
1. while与do while

while语句先判断条件，后执行循环，而do while首先执行循环，再判断条件是否进行下一次循环。

2. for语句

for语句是更加简洁的循环语句，大部分时候，for循环可以代替while循环和do while循环。

控制循环结构

Java语言没有提供go to语句来控制程序的跳转，提高了程序的可读性，但是降低了程序的灵活性，为了弥补这种不足，Java提供了continue和break来控制循环结构。

1. 使用break结束循环

break用于完全结束一个循环，跳出循环体。break语句不仅可以结束其所在循环，还可以直接结束其外层循环。此时需要在break后面紧跟一个标签，这个标签用于标识一个外层循环。如下：

Java中的标签就是一个紧跟着英文冒号的标识符。与其他语言不同的是，Java中的标签只有放在循环语句之前才有作用。

2. continue结束本次循环

continue与break的区别就是continue只是中止本次循环，开始下一次循环。可以接为continue是略过本次循环中剩下的语句。continue后面也可以紧跟一个标签，用于直接结束标签所标识循环的当次循环。

3. 深入数组

定义两个数组a、b，执行b=a，并不是将数组a的值赋给b，而是将b指向a，原有的b数组不变，但是此时第二个数组失去了引用，就变成了垃圾，它的值和长度都不会改变，直到彻底消失。
